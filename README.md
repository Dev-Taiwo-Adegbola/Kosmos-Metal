# KOSMOS — Metal Fabrication Website

A clean, responsive website for KOSMOS — a metal fabrication company specializing in kerosine tankers, petrol tankers, fueling/filling station canopies, and surface tanks. This repository holds the front-end client for marketing, project showcases, and contact/quote intake.

## About

KOSMOS fabricates heavy-duty fuel transport and storage solutions and designs protective canopies for fueling stations. The site presents company services, portfolio of completed projects, technical capabilities, and contact pathways for inquiries and quotes.

## Key Sections

- Home / Hero with value proposition
- Services (Kerosine Tankers, Petrol Tankers, Fueling Station Canopies, Surface Tanks)
- Portfolio / Case studies with photos and specs
- Technical capabilities and materials
- Request a Quote / Contact form
- About / Certifications / Safety
- FAQs and downloadable datasheets

## Features

- Responsive, mobile-first layout
- Project galleries with lightbox and filters
- Service detail pages with specs and download links
- Contact form with validation and optional file upload
- SEO-friendly metadata and sitemap
- Easy content management (Markdown / CMS integration option)
